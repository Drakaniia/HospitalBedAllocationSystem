# Contributing to Hospital Management System

Thank you for your interest in contributing to the Hospital Management System! We welcome contributions from the community.

## How to Contribute

### Reporting Issues
- Use the GitHub Issues tab to report bugs or suggest features
- Provide detailed information about the problem
- Include steps to reproduce bugs

### Submitting Changes
1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Add tests if applicable
5. Commit your changes (`git commit -m 'Add amazing feature'`)
6. Push to the branch (`git push origin feature/amazing-feature`)
7. Open a Pull Request

### Code Guidelines
- Follow Java naming conventions
- Write clear, readable code
- Add comments for complex logic
- Maintain the existing coding style

## Getting Started
1. Clone the repository
2. Compile all Java files: `javac *.java`
3. Run the GUI application: `java HospitalManagementGUI`

## Questions?
If you have any questions, feel free to open an issue for discussion.

Thank you for contributing!
